# 🎓 Smart Campus Resource Allocation System

A full-stack web application designed to streamline campus resource management by enabling students, faculty, and administrators to efficiently book, manage, and monitor campus resources such as classrooms, laboratories, seminar halls, and equipment.

## ✨ Features

- 🔐 Secure Authentication & Role-Based Access Control
  - Student Login
  - Faculty Login
  - Admin Access

- 📅 Resource Booking System
  - Book available campus resources
  - View booking schedules
  - Prevent scheduling conflicts

- ✅ Approval Workflow
  - Faculty/Admin approval for resource requests
  - Track booking status in real time

- 🔔 Notification Management
  - Booking confirmations
  - Approval and rejection alerts
  - Resource status updates

- 🛠 Maintenance Tracking
  - Report maintenance issues
  - Monitor resource availability
  - Track maintenance history

- 📊 Usage Analytics
  - Resource utilization monitoring
  - Booking history management
  - Campus resource insights

## 🎯 Project Objective

The Smart Campus Resource Allocation System aims to digitize and automate campus resource management by reducing manual scheduling efforts, minimizing booking conflicts, improving transparency, and ensuring efficient utilization of institutional resources.
